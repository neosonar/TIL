### Context
파이썬에서 데코레이터는 함수의 기능을 다른 함수로 감싸서 수정할 수 있게 해주는 디자인 패턴입니다. 이는 코드의 재사용성을 높이고 기능을 확장하는 데 유용합니다. 초보자가 이해하기 어려운 부분이 있기 때문에, 이를 쉽게 풀어 설명합니다.

### Core
다음은 기본적인 데코레이터의 사용 예시입니다.

```python
def my_decorator(func):
    def wrapper():
        print("Something is happening before the function is called.")
        func()
        print("Something is happening after the function is called.")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()
```

이 예제에서는 `say_hello()` 함수가 `my_decorator`로 장식되어, 함수 호출 전후에 추가적인 동작을 수행합니다.

### Insight
데코레이터는 주로 로깅, 접근 제어, 계측 등을 위해 사용됩니다. 데코레이터는 함수를 인수로 받는 함수로, 원래 함수를 새로운 함수로 감싸는 래퍼(wrapper)를 반환합니다. 이로 인해 코드가 간결해지고 기능이 분리되어 관리할 수 있습니다.
